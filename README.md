# Bank Transaction & Fraud Analytics (Power BI)

An end-to-end **Power BI analytics project** built to analyze **20,000 bank transactions** across customers, merchants, and fraud behavior for **January 2025**.

This dashboard-driven solution focuses on extracting **actionable insights** from transactional data using clean data modeling, DAX measures, and business-focused visual storytelling.
